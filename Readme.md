# Book Translator AI

Переводчик книг с использованием LLM API (OpenAI, Ollama, OpenRouter, DeepSeek и др.)

## Описание

Программа для пакетного перевода книг с помощью языковых моделей. Переводит текст постранично, сохраняя структуру и контекст.

## Возможности

- **Перевод книг** в форматах **TXT, EPUB, FB2, PDF** на русский и английский языки
- **Постраничный перевод** — лучший контекст, меньше запросов к API
- **Три режима перевода** — автоматический, интерактивный, гибридный (см. раздел «Режимы перевода»)
- **Редактирование перевода** — можно править отдельные страницы
- **Экспорт в Markdown, EPUB, PDF и FB2** — сохранение переведённой книги
- **Горячие клавиши** — Space (пауза), Esc (стоп), Ctrl+Enter (далее)

## Режимы перевода

Перед запуском на вкладке **"Перевод"** выбирается режим. Все режимы переводят книгу
постранично, сохраняя контекст, но по-разному подходят к проверке результата:

- **Автоматический (auto)** — переводит все страницы подряд без остановок.
  Ревью не требуется: ты видишь только готовый результат в конце. Самый быстрый
  режим, если доверяешь модели.
- **Интерактивный (interactive)** — после каждой переведённой страницы перевод
  приостанавливается для ревью. Можно принять перевод, отредактировать его вручную
  или запросить перефразировку, и лишь затем начинается следующая страница.
  Полный ручной контроль, но больше действий со стороны пользователя.
- **Гибридный (hybrid)** — внутри одной главы страницы переводятся автоматически
  подряд, а пауза на ревью ставится на **границе глав** (на последней странице главы).
  Так ты проверяешь по одной главе целиком, не останавливаясь после каждой страницы,
  — заметно меньше пауз, чем в интерактивном режиме.

> Примечание: паузы в гибридном режиме привязаны к главам книги. Если главы короткие
> (по одной странице) или не определились, гибридный режим будет выглядеть почти как
> интерактивный.

## Установка

```bash
# Клонировать репозиторий
git clone https://github.com/RobberBobFl/book-translator-ai.git
cd book-translator-ai

# Установить зависимости
uv sync

# Запустить программу
uv run python main.py
```

## Использование

1. Запустите программу: `uv run python main.py`
2. На вкладке **"Настройки"** добавьте провайдера (API ключ, Base URL)
3. Выберите модель для перевода
4. На вкладке **"Книга"** загрузите файл книги (TXT, EPUB, FB2 или PDF)
5. На вкладке **"Перевод"** нажмите "Старт"
6. После завершения экспортируйте результат (Markdown, EPUB, PDF или FB2)

## Текущие ограничения

- Для экспорта в **EPUB/PDF** требуется pandoc (а для PDF — ещё и weasyprint); обычно устанавливается автоматически при `uv sync`

## Технологии

- Python 3.12
- PyQt6 (GUI)
- LiteLLM (универсальный интерфейс к LLM API)
- SQLite (база данных)
- uv (менеджер пакетов)
